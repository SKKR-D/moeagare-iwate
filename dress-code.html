服装・ゼッケン規定

適用期間：2025年9月〜2026年8月

1服装規定

競技中及び開会式・閉会式・表彰式では、次のことを原則とする。

  1. ユニフォームは、（公財）日本ソフトテニス連盟の着用基準を満たしたものとする。ただし、学校指定の体操服は可とする。
  2. サポーター、テーピング、アンダーシャツ、アンダータイツ、アームカバー等の使用や着用を認める。
  3. シューズは、テニスシューズを着用する。ただし、体育館では室内用シューズに履きかえる。
  4. キャップやサンバイザーなどを着用しても良い。
  5. 岩手県中学校体育連盟ソフトテニス専門部服装・ゼッケン規定のゼッケンを付ける。
  6. 監督・外部指導者（コーチ）は、上下ともスポーツウェアを着用し、テニスシューズを履く。ただし、体育館では室内用シューズに履きかえる。

【服装・用具等についての補足】

  1. 服装・用具は新たにデザインを入れたり、装飾をしたりしないこと。ただし、端の部分に小さく「チーム名・名前」を記入してもよい。
  2. 靴や靴紐の左右などを意図的に色違いにしない。
  3. シャツの袖をまくったり、ズボンを下げて履いたりしない。
  4. アンダーシャツ・アンダータイツ・アームカバー（ステッチも含めて）、サポーター、テーピングを使用する場合、その色は白・黒・ベージュの単色とする。なお、アンダーシャツ・アームカバーとアンダータイツの両方を使用する場合は、同色とする。
  5. 選手の腕などへのペイント・文字の書き込み、磁気ネックレス、装飾品（ミサンガなど）、腕時計の使用は禁止する。
  6. 特別な医療用具を使用するときには、所定の医療用具届を大会本部に提出する。

2ゼッケン規定

  1. 背中のゼッケンを以下に示すとおり作成し、背中の中央に付けること。ただし、四隅を止めるものとする。
  2. 大きさは、B5判横（縦：約18cm、横：約26cm）の大きさの白地の布などに、都道府県名・姓・チーム名を記述する。
  3. シャツそのものへのプリントは認めない。
  4. ゼッケンの文字は、「漢字」、「ひらがな」、「カタカナ」等を使用し、文字は「黒」とする。

様式は、（公財）日本ソフトテニス連盟で示されている三段とし、ゼッケンの文字の位置は例のとおりとする。

岩　手
吉　田
乙部中
１/４　都道府県 ２/４　姓 １/４　チーム名
B5判横（縦 約18cm × 横 約26cm）・白地・黒文字・四隅を止める

【記載ルール】

  1. 1段目：都道府県名の「都・府・県」の文字は付けない。
  2. 2段目：同じチームに同姓の選手がいる場合は、名前の1文字目などを付け足し、区別がつけられるようにする。

＊学校の場合は「○○中」・「○○学園」など学校名がわかるように表記する。ただし「○○」のみ不可とする。また、学校名をわかりやすくするために「盛岡第二中」のように学校所在地を付記してもよい。地域クラブの場合は、所属がわかるように表記する。
